# Nexus Adaptive Detail

Nexus Adaptive Detail is planned but not enabled in this pass.

It must remain a project-specific experimental detail enhancer, not a CAS/RCAS/NIS label. A valid implementation should build a luma-detail confidence mask from gradients, local variance, min/max range, and compression-risk heuristics, then apply conservative luma enhancement with anti-ringing clamps.

Default state should remain disabled until all three samples are validated in motion.
